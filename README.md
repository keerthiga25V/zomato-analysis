# Zomato Restaurant Analysis 🍕

## Project Overview
Analyzed 148 Zomato restaurants using 
Python Pandas and Power BI.

## Tools Used
- Python (Pandas)
- Power BI

## Key Findings
- 60% restaurants have no online ordering
- Budget restaurants (300rs) most popular
- Dining has lowest rating despite being most common
- Onesta is most popular restaurant by votes

- zomato_clean.csv - Cleaned dataset
